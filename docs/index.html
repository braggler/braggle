Use Wordle skills to learn Scrabble words

Try and guess the secret word. The secret word is selected at random from allowed Scrabble words.

You may choose the length of the words. There are:

  • 1,065 three letter words,
  • 3,997 four letter words, and
  • 24,004 seven letter words.

Clues:

  • A is in the correct position,
  • B is in the word, but not at this position, and
  • C is not in the word.

Note that if a guess contains the clues A and A, it may be that A only appears once in the secret word.
